Tips for running this program

DashApp

This requires Dash, matplotlib, plotly, scikit-learn, numpy, pandas, scipy to run

The pages do take a long time to load

When running this in Pycharm, the address it runs on is http://127.0.0.1:8050/

Both DataVisualize and DashApp require the Data.Gov csv dataset to run
It can be found here:https://catalog.data.gov/dataset/authorizations-from-10-01-2006-thru-12-31-2022

To download: https://img.exim.gov/s3fs-public/dataset/vbhv-d8am/Data.Gov+-+FY25+Q1.csv
